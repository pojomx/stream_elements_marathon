Los comandos son configurables en los paneles laterales de stream elements. 

Agregado comando para setear tiempo: !settime minutos
Agregado comando para pausar tiempo: !pausetime
Agregado comando para continuar tiempo: !resumetime

los comandos son configurables

BUGS CONOCIDOS:

Al pausarlo y ponerle tiempo extra (con subs o con !addtime), el tiempo mostrado es incorrecto, se quitan como 3 segundos cada adición de cualquier cantidad de tiempo.

Se podría "terminar" el tiempo si lo pausas y transcurre el tiempo restante.  (Falta probarlo otra vez)
